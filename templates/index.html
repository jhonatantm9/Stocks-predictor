La aplicación:

Esta aplicación realiza predicciones sobre las acciones de una empresa con base en diferentes proporciones de datos de la misma. Se utiliza un modelo previamente entrenado que permite dar un resultado con base en los datos ingresados por el usuario. La respuesta de la predicción es el rendimiento predicho expresado en notación decimal.

Para la creación del modelo que permite las predicciones se utilizó la base de datos SimFin que contiene diferentes características de empresas estadounidenses.
